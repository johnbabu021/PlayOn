## Introducing PlayOn

 PlayOn - A web music player build with NextJS .Songs relay on spotify.PlayOn lets you shuffle playlist ,moods ,generes etc..




## Features

- Volume controls
- Open source
- Multi Platform
- EarPhone configuration selection
- Self Hosting
- Minimalistic
- Ad-free
